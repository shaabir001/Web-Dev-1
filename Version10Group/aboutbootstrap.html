What is Bootstrap?

Bootstrap in a Nutshell



Summary


Bootstrap is a framework created by a web-designer who formerly worked for Twitter. The main function of Bootstrap is to create a responsive website and has many different functions as seen below. You can find other Bootstrap features online or here.


How Does it Work?


A framework (like Bootstrap) comes with it's own predefined set of CSS rules and styles. Importing it allows you to "call upon" the framework inside your code. For example, this website calls upon Bootstrap in the black header (you can take a look using the Inspect tool [ctrl + i]).
